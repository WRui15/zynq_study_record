# zynq_study_record

This repository is used to record code, notes and debugging issues during the Zynq / FPGA learning process.

## Current project

### uart_adc_ram_fifo

ADC data acquisition and storage system mainly includes:

- ADC128S102 data acquisition
- RAM / FIFO data cache
- UART serial port transmission
- Vivado simulation and debugging records

## Learning content

- Verilog state machine design
- Vivado IP core usage
- RAM / FIFO timing understanding
- UART communication
- ADC SPI timing driver
